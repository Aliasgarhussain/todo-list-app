# To-Do List App

A full-stack task management application I built using Node.js, Express, MongoDB, and React.
netlify : https://todolistreactify.netlify.app
render : https://to-do-list-backend-47f2.onrender.com

## What It Does

This app lets you create, edit, and manage your daily tasks. You can add descriptions, set priorities, assign due dates, and track task status from pending to completed. It includes search and filter features to help you stay organized.

## Tech Stack

- **Backend:** Node.js + Express + MongoDB Atlas
- **Frontend:** React + Vite
- **API Testing:** Thunder Client
